# Stock Analysis App
This app is used for performing analysis on stocks

## Requirements
You will need a Google Cloud API key to retrieve stock data from Google Finance.

1) Go to https://console.developers.google.com/
2) Click on `Enable APIs and services`
3) Search for `Custom Search`
4) Click into the only result that appears
5) Click on `Enable` to activate the API
6) On the left hand toolbar, click on `Credentials`
7) Click on `Create Credentials` -> `API Key`
8) Give a name to your `API Key` and click `Create`
